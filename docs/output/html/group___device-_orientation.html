BOOL NIIsSupportedOrientation ( UIInterfaceOrientation  orientation)

For use in shouldAutorotateToInterfaceOrientation:

On iPhone/iPod touch: Returns YES if the orientation is portrait, landscape left, or landscape right. This helps to ignore upside down and flat orientations.

On iPad: Always returns YES.
