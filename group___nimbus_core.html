Detailed Description

Nimbus' Core library contains many often used utilities.

The Nimbus Core sets the foundation for all of Nimbus' other libraries. By establishing a strong base of helpful utility methods and debugging tools, the rest of the libraries can benefit from this code reuse and decreased time spent re-inventing the wheel.

In your own projects, consider familiarizing yourself with Nimbus by first adding the Core and feeling your way around. For existing projects this is especially recommended because it allows you to gradually introduce concepts found within Nimbus.
