NIReadFileFromDiskOperation Class Reference
Operations

Overview

An operation that reads a file from disk.

Provides asynchronous file reading support when added to an NSOperationQueue.

It is recommended to add this operation to a serial NSOperationQueue to avoid overlapping disk read attempts. This will noticeably improve performance when loading many files from disk at once.
